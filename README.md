# PERSONAL CA

PERSONAL CA is a Python-based Personal Finance Management System designed to help users track income, expenses, savings, and predict future expenses using Machine Learning techniques.

The system works like a personal Chartered Accountant by analyzing financial data, calculating monthly balances, and visually indicating financial health.

## Project Objectives

- Track personal income and expenses efficiently
- Analyze monthly financial data
- Predict future expenses using Machine Learning
- Help users plan savings and manage budgets
- Provide clear visual insights into financial health

## Features

- Multiple income sources (Salary and Other Income)
- Expense tracking across categories:
  - Rent
  - Food
  - Transport
  - Utilities
  - Entertainment
  - Miscellaneous
- Automatic calculation of:
  - Total Income
  - Total Expense
  - Monthly Balance
- Machine Learning-based expense prediction using Linear Regression
- Month-wise real-time data entry
- Graphical visualization:
  - Green bars indicate positive balance
  - Red bars indicate negative balance
- Future month expense and balance prediction

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
- Machine Learning (Linear Regression)
